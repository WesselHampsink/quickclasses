The QuickClasses export makes it possible to use the QuickClasses as es6 modules.

To get the best possible performance when loading your javascript, importing the QuickClasses into your module is recommended. See the example below on how to use it.

<script type="module" async> import { QuickFilter, QuickPagination, QuickSorting } from './QuickClasses.export.min.js'; new QuickPagination({...}); new QuickFilter({...}); new QuickSorting({...}); </script>
